# PlainID LangChain and LangGraph Authorizer Demo

Enterprise Authorization for LangChain, LangGraph & AI Applications

## Overview

This demo showcases PlainID's LangChain and LangGraph Authorizer with four security guardrails:

1. **Categorizer** - LLM-based prompt classification and authorization
2. **MCP Tool Control** - Policy-based agent tool authorization
3. **RAG Control** - Metadata-based document filtering  
4. **Anonymizer** - Dynamic output redaction based on user permissions

## Features

- Interactive role-based demonstration
- Industry-specific prompt scenarios (Asset Management, Healthcare, Technology)
- Dynamic mode indicator (RAG Pipeline vs Agentic Workflow)
- Side-by-side comparison with unsecured AI chat
- Real-time authorization visualization
- Context-aware access control with tiered tool permissions
- Follow-up conversation threading
